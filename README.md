# Jaspal Chauhan (Resume)
[![CI](https://github.com/jaspalch/resume/actions/workflows/deploy-resume.yaml/badge.svg)](https://github.com/jaspalch/resume/actions/workflows/deploy-resume.yaml)

This repo contains [my LaTeX-based resume](https://jaspalch.github.io/resume/) generated from the [Awesome CV template by posquit0](https://github.com/posquit0/Awesome-CV).

After a push to the `main` branch, a GitHub Actions workflow builds the resume and deploys it to GitHub Pages.
